실시간으로 KBO 기록실 정보를 크롤링 !
간편하고 직관적인 UI / UX

KBO 사이트에서 힘들게 검색했던 불편함을 "몸 쪽 꽉찬 공" 으로 해결해요 !
필터 기능으로 다양한 정보를 확인하고 승부를 예측해 꽁짜로 경품을 받아가요 !

경기 일정 및 승부 예측
몸 쪽 꽉찬 공의 메인 기능 !

한달 동안의 경기 일정을 확인해요. 또한 오늘 있을 경기의 결과를 맞춰 응모권을 획득해요. 경기 일정도 확인하고 누가 이길지 맞추는 재미까지 !

실시간 구단 순위

빠르고 간편하게 구단들의 순위를 확인해요. 깔끔한 디자인 속에 직관적인 UI 를 통해 팀의 전력을 분석해요.

실시간 선수 순위
타자, 투수를 다양한 필터를 통해 !

단 한번의 터치로 보고 싶은 필터를 선택하세요. 실시간 정보를 자신의 입맛에 맞게 받아 볼 수 있어요. 팀의 전력을 분석하여 다음 경기의 승리팀을 맞춰볼까요 ?

마이 페이지
응모권으로 경품 획득을 노려봐요 !

매주 바뀌는 경품을 자신만의 전략을 통해 응모해요. 당첨된 사람의 카카오톡 또는 핸드폰 sms로 전달해드립니다.

몸 쪽 꽉찬 공만의 차별화된 기술력을 소개합니다.

프론트 엔드, 퍼블리셔, 백 엔드 개발의 깔끔한 조화
실제 서비스와 견주어 봐도 손색없는 완벽함

  • 직관적인 UI/UX

    따로 학습이 필요없는 직관적인 유저입장의 인터페이스를 제공합니다. 최대한 야구장의 느낌을 살리기 위한 디자인을 했습니다.

  • 안정성이 뛰어난 서버와의 연동

    서버에서 제공하는 정보를 Reliable 하게 Application과 연동합니다. 단 하나의 오류의 발생도 허용하지 않은 <몸 쪽 꽉찬 공>입니다 !

  • 보안까지 신경 쓴 백그라운드 서버

    애플리케이션과 Request, Response를 주고받을 수 있는 REST API 서버를 구축했습니다. 사용자의 고유주소를 사용해 회원관리 효율적인 관리가 가능합니다.

  • 자동화로 인한 업무 효율화

    국내 KBO 서버를 백그라운드 작업을 통해 실시간으로 크롤링해와 서버에 저장합니다. 또한 예측 결과를 매일 확인하여 응모권을 지급합니다.
